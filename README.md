# Mis Intereses - Interest Tracker

A web-based application designed to log and visualize sudden interests that capture your attention. This tool helps you track what sparks your curiosity, inspired by the YouTube video "Career strategies for people with too many interests" by Unordinary Mind.

## Features

- **Interactive Bubble Map**: Visualize interests as draggable, resizable bubbles with physics-based connections
- **List View**: Browse all interests in a sortable table format
- **Monthly Timeline**: View activity organized by months
- **Multi-User Support**: Create and manage multiple user profiles
- **Dark/Light Mode**: Toggle between themes for comfortable viewing
- **Export/Import**: Save and load data as JSON files
- **Categories and Colors**: Organize interests with custom categories and color coding
- **Voting System**: Rate interests with votes
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Open `app_intereses_bugfix.html` in a modern web browser
2. Select or create a user profile
3. Choose your preferred language (Spanish/English)
4. Start adding interests as bubbles on the map
5. Use the navigation tabs to switch between Map, List, and Months views
6. Save your changes regularly
7. Export your data for backup
